# Manual GitHub repo setup

1. Create a new empty GitHub repository (no README/license/gitignore auto-generated).
2. Copy the entire `evo_skill_repo/` folder contents into your local clone root (or rename it to the repo root).
3. Commit and push.

If you want the skill available *project-locally* in another repo, copy these folders into that project’s root:

- `.claude/`
- `.codex/`

Or install by copying `skills/evo/` into the appropriate global skills directory for your CLI.
